Connect Four API Documentation

This API is for the Connect Four game. The game simply implements Connect Four for two players. When you include connectfour.js a global variable is created called connectfour. All methods should be called on this object. For example, we can make a move by doing connectfour.makeMove("R1C2") Below you can find the API:

Method NameParametersReturn ValueDescription
makeMove(cell)cell: specifies in which cell the move should be made. It should be specified as fxy, where x is a number and refers to the row, and y is also a number and refers to the column.NoneThis method allows to make a move. The moves will automatically alternate the player. The game ends when one of the players wins or it's a draw.
clearGrid()NoneNoneClears the playing grid and starts a new game.
listen(cb)cb: is the callback function. This function will be called when the grid changes. The grid is passed to the callback functionNoneThe listen(cb) functionality is used to track change in the model and receive notifications. This allows the UI to be updated when the model changes.
